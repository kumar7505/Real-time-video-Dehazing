About Our Project

Our project is an AI & ML-based dehazing system designed to enhance visibility in images, videos, and real-time camera feeds affected by fog, haze, or atmospheric disturbances. It utilizes deep learning techniques to remove haze and restore clear images and videos effectively.

Key Features & Capabilities

AI-Powered Dehazing Algorithm

  • Uses deep learning and machine learning models to remove haze from images and videos.
  • Enhances contrast and restores natural colors.

Three Modes of Operation

  • Image Dehazing: Upload an image to remove haze.
  • Video Dehazing: Process a video file to enhance visibility.
  • Real-Time Dehazing: Use a live camera feed to perform real-time dehazing.

Technology Stack

  • Frontend: HTML, CSS, JavaScript (with animations)
  • Backend: Flask (Python-based server)
  • Machine Learning Models: TensorFlow / PyTorch
  • Computer Vision Libraries: OpenCV, NumPy
  • Deployment: Flask for web hosting, integrated with AI models

Future Enhancements

  • Implement AI-based adaptive dehazing for different weather conditions.
  • Add night vision enhancement for low-light and foggy environments.
  • Improve real-time processing speed using deep learning optimization.
  • Deploy as a mobile app for on-the-go dehazing solutions.
